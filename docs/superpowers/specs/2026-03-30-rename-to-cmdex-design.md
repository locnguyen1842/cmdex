# Rename Project: Commamer -> Cmdex

**Date:** 2026-03-30
**Status:** Approved

## Summary

Rename the project from "Commamer" to "Cmdex" (short for "command index/codex"). Update all code references, branding assets, and configuration. Design three logo concepts for the user to choose from.

**Tagline:** Your command codex

## Name Conventions

| Context | Value |
|---------|-------|
| Display name | Cmdex |
| CLI binary | cmdex |
| Data directory | ~/.cmdex/ |
| Database file | cmdex.db |
| Temp file pattern | cmdex-*.sh |
| localStorage prefix | cmdex- |

## Logo Concepts

All three concepts share:
- Rounded square icon (1024x1024 master asset)
- Dark background matching app theme (#0F0F14)
- Terminal `>_` prompt motif
- Must be legible at 16x16 (favicon) through 1024x1024 (app icon)

### Concept A: Terminal Prompt Monogram
- Bold stylized **C** with integrated `>_` cursor to its right
- Accent: electric blue glow (#4A9EFF) on the prompt cursor
- Monochrome-friendly
- Most minimal, immediately reads as "dev tool"

### Concept B: Command Codex Card
- Floating index card/codex page centered on dark background
- Card has terminal header bar with `>_` prompt
- Below header: subtle lines representing stored commands
- Accent: warm amber/gold (#F0A030) for card border
- Most narrative — visually explains what the app does

### Concept C: Cmd Diamond
- Geometric diamond shape centered on dark background
- `>_` prompt symbol centered inside the diamond
- Accent: gradient from cyan (#00D4FF) to purple (#8B5CF6)
- Most abstract and modern, scales best at small sizes

**Logo delivery:** User will select a concept and provide or generate the final asset externally. The implementation plan covers replacing the icon files once the asset is ready.

## Data Migration

Users with existing `~/.commamer/` data directory need a migration path:
- On first launch, if `~/.cmdex/` does not exist but `~/.commamer/` does, copy `commamer.db` to `~/.cmdex/cmdex.db`
- Log a message indicating migration occurred
- Do NOT delete the old directory automatically (user can clean up manually)
