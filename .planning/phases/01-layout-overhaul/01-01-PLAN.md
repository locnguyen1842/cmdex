Exact requirements:
1. Add a `useRef<ReturnType<typeof setTimeout>>` called `resizeTimerRef` for debounce cleanup.
2. Add a `useEffect` (depends on `[collapse, storageKey]`) that:
   - Defines `handleResize` as a function that debounces at 100ms:
     ```typescript
     const handleResize = () => {
       clearTimeout(resizeTimerRef.current);
       resizeTimerRef.current = setTimeout(() => {
         if (window.innerWidth <= 600) {
           collapse();
         }
       }, 100);
     };
     ```
   - Calls `window.addEventListener('resize', handleResize)` on mount.
   - Returns a cleanup function: `() => { clearTimeout(resizeTimerRef.current); window.removeEventListener('resize', handleResize); }`.
3. Do NOT auto-expand when window widens — per D-01 interaction contract: "Does NOT auto-expand — user must expand manually."

   - Change `aria-label` from `"Expand panel"` to `"Expand sidebar"` (per UI-SPEC copywriting: "Expand sidebar").
