Package: `package migrations`
Imports: `"database/sql"`, `"fmt"`

CRITICAL RULES for this file:
1. Set `DisableFKDuringMigration: true` — this is the ONLY migration with this flag set to true
2. Do NOT issue `PRAGMA foreign_keys = OFF` or `PRAGMA foreign_keys = ON` inside Up or Down — the Phase 9 runner does this on the raw connection based on the flag
3. Do NOT include `UPDATE schema_version` — runner handles version tracking

`Up` function: Port db.go lines 274-311 stmts slice. The 11 statements are:
1. CREATE TABLE IF NOT EXISTS commands_new with `title TEXT` (nullable), `description TEXT` (nullable), `position INTEGER NOT NULL DEFAULT 0`, `category_id TEXT DEFAULT NULL`, `FOREIGN KEY (category_id) REFERENCES categories(id) ON DELETE SET NULL`
2. INSERT INTO commands_new SELECT with CASE WHEN transform: `CASE WHEN description = '' THEN NULL ELSE description END`
3. DROP TABLE commands
4. ALTER TABLE commands_new RENAME TO commands
5. `INSERT INTO commands_fts(commands_fts) VALUES('rebuild')`
6. `DROP TRIGGER IF EXISTS commands_ai`
7. `DROP TRIGGER IF EXISTS commands_ad`
8. `DROP TRIGGER IF EXISTS commands_au`
9. CREATE TRIGGER commands_ai — uses `COALESCE(new.title, '')` and `COALESCE(new.description, '')` (nullable columns require COALESCE)
10. CREATE TRIGGER commands_ad — uses `COALESCE(old.title, '')` and `COALESCE(old.description, '')`
11. CREATE TRIGGER commands_au — uses COALESCE for both old and new rows

Error wrapping: `return fmt.Errorf("migration 0005 up: %w", err)`

`Down` function: Reverse — recreate commands with `title TEXT NOT NULL DEFAULT ''`, `description TEXT NOT NULL DEFAULT ''`. Port from PATTERNS.md lines 340-385. The 11 statements follow the same structure but use NOT NULL columns and `COALESCE(title, '')` in the INSERT to convert nulls back to empty strings. FTS rebuild and all three trigger recreations required (same COALESCE trigger form since we're rebuilding with the same trigger code).

NOTE on Down trigger form: Down recreates the pre-v5 schema (NOT NULL columns), but the triggers should still use COALESCE for defensive correctness. Follow the PATTERNS.md Down exactly.
